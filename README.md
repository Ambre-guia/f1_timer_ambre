
# F1 timer

Application de timer de type F1 par compte Ã  rebours avec back et front

## Installation

Installer mon projet avec npm

```bash
  npm install f1_timer_ambre
  cd f1_timer_ambre
```
    
## Features

- Login / register
- Historique
- F1 Timer


## Tech Stack

**Client:** React, Jest, Linter, Prettier

**Server:** Node, Express, Jest, Linter, Prettier


## Running Tests

Pour lancer les tests en front

```bash
  cd ./frontend
  npm run test
```

Pour lancer les tests en back

```bash
  cd ./backend
  npm run test
```
